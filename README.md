# Analytics Systems APIs

Набор прослоек для работы с API систем аналитики:

 - Google Analytics
 - Yandex.Metrika
 - Appmetrica

## Начало работы 

```bash
git clone https://github.com/ArticsIS/Analytics-Systems-APIs
cd Calltracking-Systems-APIs
python3 -m pip install -r requirements.txt
python3 example.py
```
